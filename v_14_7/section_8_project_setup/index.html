Nodejs and WebPack

Use this to include babel and react and everything needed

  1. Node Js - Nodejs.org
  2. package.json - is your project plan/blueprint, has dependencies/libraries, says what we need
  3. npm - repo of modules
  4. Webpack - webpack is a module loader, minifies your imported files, compile sass (Gulp/Grunt do similar things), seems to be go to for react development
  5. Webpack config - webpack.config.js - settings for webpack, somewhat complicated in ways, will use basic setup will be used to get started
    • use babel
    • transform code
    • which file to load our code from
    • which file we want webpack to output or code to
  6. .babelrc - config file for babel
  7. Get the bundle.js file locally
    1. install webpack globally -> yarn global add webpack@1.13.0
    2. run -> webpack
    3. will output bundle.js file to project directory to put on server
  1. Using yarn - yarn init - setup project, starts/adds a package.json
  2. install react v-15.0.2 ---> yarn add react@15.0.2
  3. install other project dependencies (react/babel) ---> yarn add react-dom@15.0.2 webpack@1.13.0 webpack-dev-server@1.14.1 babel-core@6.8.0 babel-loader@6.2.4 babel-preset-es2015@6.6.0 babel-preset-react@6.5.0 babel-preset-stage-1@6.5.0
  4. Be sure to gitignore the node_modules directory -> **node_modules/
